Icon-only button for toolbars, card navigation (back/skip), and dismiss actions. Always provide `label`.

```jsx
<IconButton label="Previous card" variant="outline" round><ChevronLeft/></IconButton>
<IconButton label="Close" variant="ghost"><X/></IconButton>
```

Variants: `ghost` · `solid` · `outline`. Use `round` for the circular swipe-navigation controls in the app.
